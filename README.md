# ⛪ Iglesia ni Chancellor

> ⚠️ **FOR FUN ONLY** — This is a parody/practice project built for design and development purposes. "Iglesia ni Chancellor" is a fictional entity created solely as a coding exercise and is not affiliated with, endorsed by, or representative of any real religious organization.

**🔗 Live Site:** [iglesia-ni-chancellor.vercel.app](https://iglesia-ni-chancellor.vercel.app)

A modern, minimalist landing page for a fictional church — built with plain HTML, CSS, and JavaScript, featuring a warm cream-and-gold design system and smooth scroll animations.

---

## ✨ Features

- **Fully responsive** single-page layout (mobile, tablet, desktop)
- **Scroll-triggered animations** powered by [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Sticky navbar** with scroll-based background toggle and mobile hamburger menu
- **Sections included:**
  - Hero with call-to-action buttons
  - About / Our Story with mission values
  - Service Times schedule
  - Church benefits/perks grid
  - Upcoming Events
  - Community testimonials
  - Contact section with embedded Google Map and validated contact form
- **Accessible markup** — semantic HTML5, skip link, ARIA labels, keyboard-navigable nav
- **Zero build step** — no frameworks, no bundlers, just open `index.html`

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties, Flexbox/Grid) |
| Interactivity | Vanilla JavaScript |
| Animations | [AOS](https://michalsnik.github.io/aos/) (CDN) |
| Hosting | [Vercel](https://vercel.com) |

## 🚀 Running Locally

No build tools required — this is a static site.

```bash
git clone https://github.com/<your-username>/iglesia-ni-chancellor.git
cd iglesia-ni-chancellor
```

Then simply open `index.html` in your browser, or serve it locally:

```bash
# using VS Code Live Server extension, or:
npx serve .
```

## 🌐 Deployment

Deployed on **Vercel** as a static site — no configuration needed beyond pointing Vercel at the repo root.

## 🎨 Design

- **Palette:** cream/off-white base with warm gold accents
- **Typography:** serif headings for a traditional, established feel paired with a clean sans-serif body font
